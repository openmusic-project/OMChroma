SMPL-A2

CLASS SLOTS:
- elts number of elements (components) for the event [int] [default = 1]
- action-time start time of the whole event [sec] [default = 0]
- user-fun a lambda patch or function to process internal elements at synthesis time [lambda/fun-name] [default = nil]

Optional:
- precision float precision in the Csound score [default = 4]

CSOUND PARAMS (KEYWORDS):
- e-dels action time [sec] [default = 0.0]
- durs duration [sec] [default = 1.0]
- amp maximum amplitude [linear (0.0,1000.0) or dB (-INF,0.0)] [default = 0.0]
- f0 min transposition factor [1=same as original] [default = -2.0]
- f0max max transposition factor [1=same] [default = -1.0]
- f0env envelope for the transposition factor [GEN] [default = (:cs-table gen07 :points ((0 0) (32768 100) (65536 0)))]
- f0jta jitter amplitude for the transposition factor [0-1] [default = 0.1]
- f0dur duration of the transposition envelope [sec] [default = 1.0]
- jta jitter amplitude for the max amplitude [0-1] [default = 0.5]
- afil file name [int, sound, string, pathname or GEN] [default = "santuri.aiff"]
- skip starting point in file [%] [default = 0.0]
- aenv amplitude envelope [GEN] [default = (:cs-table gen07 :points ((0 0) (65536 100)))]
- win duration of the local attack/decay [sec] [default = 0.01]
- wrap wrap [0: locations beyond the file produce silence / other: locations wrapped to the duration of the sound] [default = 1]
