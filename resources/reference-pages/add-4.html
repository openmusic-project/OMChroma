ADD-4

CLASS SLOTS:
- elts number of elements (components) for the event [int] [default = 1]
- action-time start time of the whole event [sec] [default = 0]
- user-fun a lambda patch or function to process internal elements at synthesis time [lambda/fun-name] [default = nil]

Optional:
- precision float precision in the Csound score [default = 4]

CSOUND PARAMS (KEYWORDS):
- e-dels action time [sec] [default = 0.0]
- durs duration [sec] [default = 1.0]
- amp maximum amplitude [linear (0.0,1000.0) or dB (-INF,0.0)] [default = -6.0]
- freq frequency [Hz] [default = 440.0]
- aenv amplitude envelope (GEN table) [default = (:cs-table gen-07 :points ((0 0.0) (1000 1.0) (3000 1.0) (4096 0.0)))]
- fenv vibrato envelope (GEN table) [default = (:cs-table gen-07 :points ((0 440.0) (4096 880.0)))]
- atk attack time of the amp envlp [sec] [default = 0.01]
- dec decay time of the amp envlp [sec] [default = 0.01]
- phs initial phase [rad] [default = 0.0]

DESCRIPTION:

=============================================================================
ADD4.ORC
SIMPLE ADDITIVE SYNTHESIS ADAPTED TO READ PARTIAL ANALYSIS DATA / MONO
AMPLITUDE ENVELOPE WITH OSCILI AND ATTACK/RELEASE VALUES
=============================================================================

Timbre: simple additive synthesis with variable amplitude and frequency
Synthesis: additive same units
Coded: ms 17/07

This class reads absolute values for amplitudes and frequencies
coming from analysis data. In this case, set freq to 1.0 (scaler) and amplitude
to 1000.0 or 0.0 (max dB). Note that GEN functions should have a negative number
in order not to be rescaled (GEN 0 -7)
It can also be used with normalized amp and freq functions, and, in this case,
amp and freq should have a reasonable value.
